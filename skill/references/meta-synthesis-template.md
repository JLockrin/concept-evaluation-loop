## Chat delivery rules



- State mode upfront: *"Evaluated 3 approaches to [problem]. Recommendation: …"*

- Lead with **Recommendation** and **TL;DR**.

- Keep chat body under ~300 words.

- Link to `meta-synthesis.md` and each option's synthesis — not every pro/against file unless the user asks.

- Do not re-run full arguments in chat — compare and point to files.
